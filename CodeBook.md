### Introduction
This project has a script that takes data from the "Human Activity Recognition Using Smartphones Data Set"
clean up the data and generates a unique row of data for each subject of the experiment for each activity they perform


### Output
The output of the running the script is a CSV file called "tidyData.txt" that contains the following variables per observation.

The first variable identifies the subject of the experiment
The second variable is for the activity performed by the subject
The rest are 66 columns of various readings for that subject while performing that activity.
